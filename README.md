Happy Guest lambda tester
===================================================

## :star: Features
This repository helps to test lambda functions locally


## :bulb: Before installation
Login into npm to get package permissions.
```
npm login
```

## :cloud: Installation
```
# Enter into folder function
npm i @happyguestmx/tester
```

##  :pencil: Example
```
# Enter into folder function
npm test [ENV] 
```
